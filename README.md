## Getting Started

### Installation

Clone the repo:
```bash
git clone https://github.com/yourusername/globetrotter.git
cd globetrotter
```

Install the dependencies:

```bash
npm install
```

### Development

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```
